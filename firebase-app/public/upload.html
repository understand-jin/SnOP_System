서버 실행 방법

python app.py

3
위 "업로드 화면 열기" 버튼 클릭
이 페이지를 새로고침하면 서버 상태가 녹색으로 바뀝니다.
업로드 파일 안내

부진재고 분석을 위해 아래 5개 파일이 필요합니다. SAP 리포트 원본 Excel/CSV 파일을 그대로 업로드하면 됩니다.

#파일명SAP 리포트주요 컬럼필수
1재고개요 (Standard) MB52 / 재고개요 자재, 자재 내역, 플랜트, 특별 재고, 저장 위치, 배치, 기말 재고 수량 필수
2자재수불부 (Cost) MB51 / 자재수불부 자재, 기말(수량), 기말(금액)합계 필수
3배치별유효기한 (Expiry) 배치관리 유효기한 리포트 자재, 배치, 배치만료일 필수
43개월매출 (Sales) SD 매출 리포트 (3개월) 년월, 자재코드, 순매출, 순매출수량 필수
5대분류_소분류 (Classification) 자재 분류 마스터 자재, 대분류, 소분류 필수
-품절 분석 (Stockout) 재고개요 + 3개월매출 재고개요 + 3개월매출 2개 파일 사용 별도
